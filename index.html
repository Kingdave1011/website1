📦 Installation Instructions

Windows Installation:

  1. Download SpaceShooter_Setup.exe from itch.io
  2. If Windows shows a warning, click "More info" then "Run anyway"
  3. Run the installer and follow the wizard
  4. Choose to create desktop shortcut (optional)
  5. Click Install and wait
  6. Launch from desktop or Start Menu!

Linux Installation:

  1. Download the Linux package from itch.io
  2. Extract the ZIP file
  3. Open terminal in the extracted folder
  4. Make executable: chmod +x "Space Shooter V2.x86_64"
  5. Run: ./"Space Shooter V2.x86_64"
  6. Play!

Tip: If the installer script is included, you can run it for automatic setup!

⚠️ Security Notice: Windows may show "not commonly downloaded" - just click "Keep" and "Run anyway". Our game is safe!
